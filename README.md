# smart-farm

Website monitoring temperature, humidity and Soil Moisture based internet of things (IoT).

## ```Hardware```
1. Esp32/Nodemcu
2. Sensor DHT22/11
3. Soil Moisture Sensor

## ```Backend```
1. Nodejs
2. Expressjs
3. JWT/random value

## ```Frontend```
1. Vue/NuxtJs
2. Vuetify
3. MomenJs
4. HTML
5. CSS
6. Javascript
7. Axios

## ```Database```
1. Mysql
2. Thingspeak
3. Firebase

## ```Tools```
1. Figma
2. Visual Studio Code
3. Xampp & phpmyadmin
4. Arduino IDE
5. Nodejs
